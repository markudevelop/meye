First launch on macOS

Meye is free and signed without an Apple Developer account, so macOS will warn the first time you open it. One-time steps:

  1. Move Meye.app to your Applications folder.
  2. Right-click → Open (not double-click).
  3. Click Open in the dialog.

If macOS says “Meye is damaged,” run in Terminal once: xattr -dr com.apple.quarantine /Applications/Meye.app

First launch on Windows

The Windows installer isn't code-signed yet, so SmartScreen may warn the first time. One-time steps:

  1. Run Meye_…_x64-setup.exe.
  2. If a “Windows protected your PC” dialog appears, click More info → Run anyway.

FAQ

Good questions

Where does my data go?

Nowhere. Meye records, indexes, and runs AI search entirely on your own machine. Nothing is uploaded — there are no accounts and no servers involved in capturing or searching your memory.

Is Meye really free?

Yes. Meye is free to download and use, with no subscription and no account required.

Is Meye open source?

Yes. Meye is fully open source under the AGPL-3.0 license — you can read every line, audit what it does with your data, and build on it. The source lives on GitHub. Meye is built on the open-source screenpipe capture engine.

Which platforms are supported?

Meye is a native macOS app today. It keeps the recorder alive with a launchd background agent so capture survives logout and reboot.
